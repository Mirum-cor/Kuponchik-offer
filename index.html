Основатель и директор — про kuponchik.by

Купончик является онлайн-сервисом, на котором предприниматели и компании размещают информацию о скидках для своих клиентов в виде купона.

Купончик — это агрегатор скидок для разных категорий товаров и услуг для более чем 120 городов нашей страны.

Рассылка

Данный функционал доступен только для клиентов, размещающих платные предложения.

Рассылка создаётся по собственной «Базе клиентов» юр. лица (т.е. по базе клиентов, которые скачали купон данного юр. лица или нажали кнопку «Жду лучшего предложения» на странице с купоном), по любым возможным критериям, например: всем женщинам старше 25 лет, только постоянным клиентам или только потенциальным клиентам и т.д.

Должно быть создано специальное скидочное предложение для клиента.

Данный функционал доступен только для клиентов, размещающих платные предложения.
